# Working-hours-calculation
From a CSV file, calculates the hours made at each check-in. 

Displays the number of hours and missing hours per week, then the total since the 1st check in.

## Installation

Tested with python3

Packages you need :

* dateutil
* pandas

```bash
sudo pip3 install python-dateutil pandas
```

## Run

Change in the python script the filename of your CSV file (line 9, "check-in.dat").
You can also change the delimiter (I used ";").

```python
# read csv file
df = pd.read_csv("check-in.dat", skiprows=[0],sep=';')
```
And then:
```bash
./calculate-hours.py
```
